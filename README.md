# Basic Webshop in Nodejs

Ein Projekt für die CS Community.
Bevor das Programm mit dem Command 'path/run' gestartet werden kann, müssen folgende Programme und Module installiert werden (npm install).

# Dependencies:

Programme:
- NodeJs
- MongoDB

Module:
- express
- morgan
- method-override
- serve-favicon
- errorhandler
- mongoose
- socket.io
